# 트잇 (Twit)
###  프로젝트 공유 게시판 사이트
- 뉴비 개발자들은 자신의 프로젝트 결과물을 다른 사람에게 노출하기 쉽지 않습니다.
- 그래서 개발자 지망생들이 서로의 결과물을 공유하고, 피드백하며 사용자 트래픽 경험까지 챙길 수 있는 프로젝트 공유 게시판을 개발하게 되었습니다.
- 공유된 프로젝트에 포스트잇을 붙이듯이 자유롭게 피드백을 남길 수 있어서 포스트잇의 트잇을 따 이름을 지었습니다.

- 배포주소
